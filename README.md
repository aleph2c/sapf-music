# SAPF-MUSIC

## Setting up the Environment

```bash
git clone git@github.com:aleph2c/sapf-music.git
cd sapf-musc
python3 -m venv venv
source ./venv/activate
pip install --upgrade pip
pip install -r requirements.txt
```
